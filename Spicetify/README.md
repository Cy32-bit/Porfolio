# Spicetify Installer

A clean, structured Windows batch installer for [Spicetify CLI](https://spicetify.app/).

## Features

- **Clean ASCII UI** — No emojis, no bloat, works in any terminal
- **Big stylized header** — Retro ASCII art logo
- **Safety prompt** — Confirms before executing remote code
- **Error handling** — Clear success/failure feedback
- **Single file** — Just one `.bat` script

## Usage

1. Download `install-spicetify.bat`
2. Double-click to run, or execute from Command Prompt:
   ```cmd
   install-spicetify.bat
