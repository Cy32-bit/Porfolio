# Movie Rater

A sleek, dark-mode desktop app for cataloging films with professional-grade metrics.

## 🛠 Installation & Usage

### 1. Requirements
* Python 3.8 or higher
* Libraries: `pandas`, `openpyxl`, `customtkinter`, `Pillow`

### 2. Automatic Build (Recommended)
Run the setup script for your OS to create a standalone executable:
- **Windows:** Run `setup_windows.bat`
- **macOS/Linux:** Run `setup_mac_linux.sh`

### 3. Manual Execution
```bash
pip install pandas openpyxl customtkinter Pillow
python movie_rater.py
